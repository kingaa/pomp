title: pomp version 6.4 released
---

**pomp** version 6.4 has been released to CRAN and will soon appear on [the CRAN mirrors](https://cran.r-project.org/mirrors.html).
This release contains only one minor feature upgrade:
the `coef` function can now extract parameters by name from lists of pomp objects (created via `concat` for example).
